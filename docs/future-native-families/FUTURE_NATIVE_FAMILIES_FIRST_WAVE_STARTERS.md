# Future Native Families: First-Wave Starters

最初に native family として起こす候補です。

- `pbd-rope`
- `mpm-granular`
- `fracture-lattice`
- `volumetric-density-transport`

各 family について starter-runtime に

- schema
- solver seed / prototype

を入れています。

目的は、後続AIが次の順で増築できることです。

1. normalized schema を確定
2. adapter を追加
3. deterministic solver step を追加
4. debug renderer を追加
5. UI パラメータ束を追加
6. verifier を追加
